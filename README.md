# IRON — Интерактивная Российская Онлайн-база компьютерного железа

IRON — это веб-каталог компьютерных комплектующих с возможностью фильтрации, сравнения и детального просмотра характеристик. Проект ориентирован на русскоязычную аудиторию DIY-сообщества.

## 🚀 Функционал

- **7 категорий товаров**: видеокарты, процессоры, материнские платы, оперативная память, SSD, блоки питания, корпуса
- **Динамическая фильтрация** по производителю, чипсету, объёму памяти, сокету, типу памяти и другим параметрам
- **Пагинация** с выбором количества товаров на странице (12, 24, 48, 96)
- **Сравнение до 4 товаров** с плавающей панелью и страницей сравнения (выделение лучших показателей)
- **Детальные страницы товаров** с иконками разъёмов питания и видеовыходов
- **Поиск товара** на Wildberries, Ozon, Avito, Google, Яндекс, YouTube
- **Форма предложения товара** с отправкой в Supabase
- **Светлая и тёмная тема**

## 🛠 Технологии

- HTML5, CSS3, JavaScript (Vanilla)
- Supabase (PostgreSQL, аутентификация, REST API)
- GitHub Pages (хостинг)

## 📁 Структура JSON-файлов

- `all GPU.json` — видеокарты
- `all CPU.json` — процессоры
- `all Motherboard.json` — материнские платы
- `all RAM.json` — оперативная память
- `all Storage.json` — SSD накопители
- `all PSU.json` — блоки питания
- `all Case.json` — корпуса
